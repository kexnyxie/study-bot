# 📚 Gyeol

Gyeol is a modern Discord productivity bot built for students who actively use discord. It is designed to help you stay focused, organize coursework, and make studying rewarding all while having in app gaming/economy features which can be used in breaktime for fun.

![status](https://img.shields.io/badge/status-in%20development-yellow)
![node](https://img.shields.io/badge/node-%3E%3D18-green)

---

## Features

- 🍅 **Custom Pomodoro Sessions** — multi-cycle focus/break timers with live progress tracking
- ✅ **Todo Lists** — add, complete, and manage personal study tasks
- 📈 **XP & Leveling** — earn XP from studying, messaging, reacting, and completing sessions
- 🎂 **Cake Economy** — daily rewards with streak bonuses
- 🎲 **Coinflip** — a lighthearted betting mini-game
- 🏆 **Leaderboards** — see who's putting in the most work
- 📊 **Study Statistics** — track daily and total focus time
- 📅 **Semester Planner** *(coming soon)*

## 🛠 Setup

```bash
git clone https://github.com/kexnyxie/study-bot.git
cd study-bot
npm install
```

Create a `.env` file:
DISCORD_TOKEN=your_bot_token
CLIENT_ID=your_application_id (the application ID available in general information)
GUILD_ID=your_test_server_id (bot server's id)

Register commands and start the bot:
```bash
node deploy-commands.js
node index.js
```
